## ADRテンプレート

```markdown
# ADR-XXX: タイトル

## ステータス
提案中 | 承認済み | 却下 | 置き換え済み

## コンテキスト
決定が必要な背景や問題を記述

## 決定事項
実際の決定内容を記述

## 理由
なぜこの決定をしたのかを記述

## 結果
この決定による影響や結果を記述
```
